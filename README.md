# ActivityFinder - Frontend (Next.js)

Dette er frontend-delen av ActivityFinder, bygget med **Next.js** og koblet til en **ASP.NET Core backend**.

## 🚀 Teknologi
- **Next.js 15** med **App Router**
- **Tailwind CSS**
- **TypeScript**
- **Azure PostgreSQL** (for backend-database)
- **ASP.NET Core** backend
